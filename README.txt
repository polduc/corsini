Nous avons travailler le projet sous le langage c++ sous l'IDE eclipse.

Les fichiers sources se trouvent dans le dossier /src.
Les headers dans /includes.

Pour pouvoir exécuter le projet : importer le projet sous eclipse, dans /src un fichier main.c contient l'ensemble des appels de toutes les méthodes implémentées.
Il suffit de le "run" et d'entrer le nom de l'instance tout court(sans extension). On precisera le nombre de partitionnement(k) en argument.

Sinon un Makefile est disponible dans le dossier /Debug.
Pareil à l'exécution du programme on precisera le nom de l'instance et le nombre de classe(k).

Le programme nous donne en sortie l'affichage suivant :
-Le nom de la méthode
-La meilleure partition
-La valeur optimale
-son temps d'exécution
